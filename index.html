Translate software using Google

How this works:

  1. You upload an untranslated gettext .PO file and select languages
  2. It's submitted in a request to Google Language API to translate it
  3. Translation is saved to a new .PO file you can download
  4. A real person (you) edits the file, fixes translation errors, and has a brand new translation. Win! All around hapiness ensues.
